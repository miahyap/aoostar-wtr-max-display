**AOOSTAR LCD**

Drives the integrated case display of the AOOSTAR WTR MAX (and GEM12+ PRO)
using the open-source aoostar-rs driver (asterctl + aster-sysinfo) — no
vendor software required. Shows CPU, memory, network and temperature stats
on the LCD, starts at boot, and is configured from
*Settings → Utilities → AOOSTAR LCD*.
